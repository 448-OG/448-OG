### Research
- [X3DH-448](https://github.com/448-OG/X3DH-448) - A modified version of Extended Triple Diffie-Hellman (X3DH) Key Agreement Library used in Signal Protocol. This swaps SHA256 for Blake3 as it is aimed a very low power hardware support even where SHA256 accelleration instructions are not available.
- [MEMSECURITY](https://github.com/448-OG/MEMSECURITY) - Securely hold secrets in memory and protect them against cross-protection-boundary readout via microarchitectural, via attacks on physical layout, and via coldboot attacks.
- [x402-Links](https://github.com/JamiiDao/x402-Links) - The x402-links is a specification for discovering and interacting with x402 resources in the real-world through deep-links and QR Codes. 
- [Blinded-Diffie-Hellman-Merkle-Algorithm](https://github.com/448-OG/Blinded-Diffie-Hellman-merkle) - A research implementation of the Blinded Diffie-Hellman Algorithm used in Ecash for financial inclusion of grassroot communities.
